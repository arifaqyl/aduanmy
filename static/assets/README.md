# TrafficMY open-source media assets

Original Malaysia geometry and station names. Schematic **style** references public transit map literature — no copyrighted operator artwork is copied or hotlinked.

## Directory layout

| Path | Purpose |
|------|---------|
| `icons/` | UI icons (Tabler MIT + TrafficMY originals) |
| `markers/` | Reusable schematic node components for `scripts/build_line_svgs.py` |
| `schematics/` | Generated line diagrams (mirror of `static/lines/` after build) |

## Build

```bash
python scripts/build_line_svgs.py
```

Writes to `static/lines/` and `static/assets/schematics/` plus `static/data/kv-system-map.svg`.

## Rules

- Do not embed TfL, MTR, or MyRapid copyrighted map artwork.
- Station order from public references (`lines-reference.json`, `line-stations.json`).
- IBM Plex Sans labels in generated SVGs (loaded by the web app, not embedded fonts).
